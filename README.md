# Travel Guide Website

Static website that provides travel information, blog posts and portfolio pages. This repository contains HTML, CSS and JavaScript assets intended to be served as a static site (for example via GitHub Pages, Netlify, or a simple static file server).

## Contents

- `index.html` - Home page
- `Home.html` - Alternate home page (if used)
- `About.html` - About page
- `Portfolio-Page.html` - Portfolio page
- `blog/` - Blog section
  - `blog.html` - Blog listing
  - `post-*.html` - Individual blog posts
  - `blog.json` - Blog metadata (used for generating lists)
- `images/` - Image assets used across the site
- `intlTelInput/` - Third-party phone input library (CSS, JS, utils)
- CSS files: `index.css`, `nicepage.css`, `About.css`, `Blog-Template.css`, `Portfolio-Page.css`, `Post-Template.css`
- JS files: `nicepage.js`, `jquery.js`, `intlTelInput.min.js`, `utils.js`
